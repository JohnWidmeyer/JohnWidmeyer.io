Iphi is a fitness application designed from the ground up to target a gap in the market-- specifically software that aims to assist in meeting fitness goals through automated programming. A fitness 'program is a routine designed to push a person's body to the limit in order to affect change on one's body; Iphi intends to assist in this by generating generic routines that can be tailored by 'favourites', while additionally providing the flexibility to create entirely custom workouts if desired. Iphi tracks the user progress and charts it against a plot of their specified goals. By marrying simple UI with a powerful back-end, Iphi aims to be user friendly without sacrificing the power and flexibility of a native website.
Over the course of a goal cycle the user will be provided a number of automatically generated workouts; upon completion the user will save their reported maxes for the main lifts and those will be charted in the user profile for visual progress. Iphi does not recommend weights for the user to lift, being intended for those with experience in the gym, Iphi aims to break up the monotony of normal fitness programs. With seamless google integration Iphi will work with and without internet access (only losing access to achievements with your lack of connection).

Easy Navigation

The 'Iphi' main menu is easy to use and straight forward, providing navigation to the automatically scheduled workout, the custom workout window, your current achievements, and your saved workouts. With simplicity in mind the main screen lays everything out.

Instant Generation

'Iphi' will automatically generate a workout based on your chosen workout split and training level. The scheduled workout for the day rotates based on completion (eg. Chest/Tri's then upon completion Back/Bi's is scheduled).

Flexibility

If the automatic generation doesn't fit your needs 'Iphi' allows you to create a custom workout as well. By swiping from the right to the left, the exercise is moved into the active side of the workout-- by saving you gain the ability to load the custom workout whenever you want.

Data Visibility

The 'profile' module allows the user to set their name, lift goals, and view their progress. By completing workouts the user's data is appended to an android file in the phones directory-- this data persists through app close and phone powercycle.
